4 3 2 1   

序列：1 2 3 4  
第一位：k = 7, k / 3! = 1, 可以得到2, 然後 k = k % 3! = 1, 再把已經拿走的2去掉

序列：1 3 4  
第二位：k = 1, k / 2! = 0, 可以得到1, 然後 k = k % 2! = 1, 再把已經拿走的1去掉

序列：3 4  
第三位：k = 1, k / 1! = 1, 可以得到4, 然後 k = k % 1! = 0, 再把已經拿走的4去掉

序列：3  
第四位：k = 0, k / 0! = 0, 可以得到3, 然後 k = k % 0! = 0, 再把已經拿走的3去掉

最後可以得到 2 1 4 3
