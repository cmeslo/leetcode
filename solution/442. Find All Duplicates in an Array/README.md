# 442. Find All Duplicates in an Array

## 442.cpp
每遇到一個數字，就把該數字對應的下標的值取負數，遇到負數代表該下標已經出現過一次。

```
原數組內容：
[ 4, 3, 2, 7, 8, 2, 3, 1]

4
[ 4, 3, 2, 7, 8, 2,-3, 1]

3:
[ 4,-3, 2, 7, 8, 2, 3, 1]

2
[ 4,-3,-2, 7, 8, 2, 3, 1]

7
[ 4,-3, 2, 7, 8, 2, 3, 1] --->找到2

8
[-4,-3, 2, 7, 8, 2, 3, 1]

2
[-4,-3,-2, 7, 8, 2, 3, 1] 

3
[-4, 3,-2, 7, 8, 2, 3, 1] --->找到3

1
[-4, 3,-2,-7, 8, 2, 3, 1]
```
