## Stack (385_02.cpp)
使用stack來做
1. 每當看到``[``代表需要創建一個新節點
2. 每當看到```,```或者```]```代表可以構迼一個元素

例子```"[123,[456,789],[]]"```

stack的變化情況
```
[]
[123]
[123], []
[123], [456]
[123], [456,789]
[123 , [456,789]]
[123 , [456,789]], []
[123 , [456,789] , []]
```
